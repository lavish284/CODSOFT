# Iris Flower Classification - CodSoft Data Science Internship

## Project Overview

This project is based on Machine Learning classification. The objective is to develop a model that can classify Iris flowers into different species based on their measurements.

The model predicts three categories:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Dataset Description

The Iris dataset contains measurements of flowers including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The dataset consists of 150 samples belonging to three different species.

## Tools and Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm Used

**K-Nearest Neighbors (KNN)**

KNN is a classification algorithm used to classify data points based on the similarity between them.

## Project Workflow

1. Importing required libraries
2. Loading the Iris dataset
3. Performing data exploration
4. Visualizing the dataset using graphs
5. Data preprocessing
6. Splitting data into training and testing sets
7. Training the machine learning model
8. Evaluating model performance

## Model Performance

The model achieved approximately 96-100% accuracy on the test dataset.

## Conclusion

The Iris Flower Classification project successfully demonstrates the use of machine learning techniques to classify flower species based on their physical measurements.

This project was completed as part of the CodSoft Data Science Internship

# Sales Prediction Using Machine Learning - CodSoft Data Science Internship

## Project Overview

This project focuses on predicting product sales based on advertising expenditure using Machine Learning techniques.

The objective is to build a regression model that can analyze the relationship between advertising platforms and sales performance, helping businesses make data-driven marketing decisions.

## Dataset Description

The dataset contains advertising expenditure information across different platforms:

- TV Advertising
- Radio Advertising
- Newspaper Advertising

The target variable is:

- Sales (Product Sales)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Project Workflow

The project follows these steps:

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Visualization
4. Feature Selection
5. Splitting Data into Training and Testing Sets
6. Building Machine Learning Model
7. Model Evaluation
8. Sales Prediction

## Machine Learning Model Used

### Linear Regression

Linear Regression algorithm was used to predict sales based on advertising investment.

The model learns the relationship between advertising expenses and sales values to make future predictions.

## Data Analysis & Visualization

The project includes:

- Distribution analysis of sales
- Relationship analysis between advertising platforms and sales
- Correlation analysis
- Visualization of important features

## Model Performance

The Linear Regression model was evaluated using performance metrics such as:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)

The model successfully demonstrates the ability to predict sales based on advertising data.

## Business Insights

1. Advertising expenditure has a significant impact on sales performance.

2. TV and Radio advertising show strong relationships with increasing sales.

3. Machine Learning models can help businesses forecast future sales.

4. Data-driven marketing decisions can improve resource allocation and business strategies.

## Conclusion

The Sales Prediction project demonstrates how Machine Learning can be applied for business forecasting.

By analyzing advertising data and training a regression model, the project provides valuable insights into sales prediction and marketing optimization.

This project was completed as part of the CodSoft Data Science Internship.
