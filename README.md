# Iris Flower Classification - CodSoft Data Science Internship

## Project Overview

This project is based on Machine Learning classification. The objective is to develop a model that can classify Iris flowers into different species based on their measurements.

The model predicts three categories:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

## Dataset Description

The Iris dataset contains measurements of flowers including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The dataset consists of 150 samples belonging to three different species.

## Tools and Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm Used

**K-Nearest Neighbors (KNN)**

KNN is a classification algorithm used to classify data points based on the similarity between them.

## Project Workflow

1. Importing required libraries
2. Loading the Iris dataset
3. Performing data exploration
4. Visualizing the dataset using graphs
5. Data preprocessing
6. Splitting data into training and testing sets
7. Training the machine learning model
8. Evaluating model performance

## Model Performance

The model achieved approximately 96-100% accuracy on the test dataset.

## Conclusion

The Iris Flower Classification project successfully demonstrates the use of machine learning techniques to classify flower species based on their physical measurements.

This project was completed as part of the CodSoft Data Science Internship
